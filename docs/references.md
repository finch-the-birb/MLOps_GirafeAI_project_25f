# Научные ссылки

## FNSPID

- [Financial News and Stock Price Integration Dataset](https://arxiv.org/pdf/2402.06698)
- Point-in-time выравнивание новостей и котировок.

## iTransformer

- [iTransformer: Inverted Transformers Are Effective for Time Series Forecasting](https://arxiv.org/pdf/2310.06625)
- Inverted embedding: attention **между variates**, не по времени.
- Реализация: `src/stock_forecaster/third_party/tslib/`

## FinBERT

- [FinBERT: A Pretrained Language Model for Financial Communications](https://dl.acm.org/doi/pdf/10.1145/3694860.3694870)
- Модель: `ProsusAI/finbert` (HuggingFace).

## RL context (будущая работа)

- [AoE-MR — Actionable out-of-distribution Expert Mixture for RL in Algorithmic Trading](https://people.ece.ubc.ca/aminmf/AoE-MR_CanadianAI2025.pdf)
- Текущий проект — supervised Phase 1; latent fusion пригоден как state для RL.
